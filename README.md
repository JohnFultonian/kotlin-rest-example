# Kotlin REST Example
## Intro
A simple REST server example using Kotlin

## Development
### Prerequisites
Just [Docker](https://www.docker.com)
### Run
`make run`

### Test
`make test`

### Build
`make build`

## Usage
Doesn't do much right now, but you can

`curl http://localhost:8080/campaign`

and

`curl http://localhost:8080/campaign/1`

if you `make run`
